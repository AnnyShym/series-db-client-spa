# series-db-client-spa
Modern Programming Platforms [2]


A simple web client for viewing and editing the TV series database.


How to start:

- the server side
1) npm install
2) npm start

- the client side (another terminal)
1) cd ./frontend
2) npm install
3) npm start
   
- browser
1) http://localhost:3000/
